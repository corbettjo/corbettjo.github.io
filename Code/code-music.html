Music Editor

Music editor was the final project for CS3500 (Object Oriented Design). It works with java's MIDI library to add, remove, and edit notes of a song. It can also play songs at different tempos, as well as play and pause.

Although it sounds simple, this project was almost completely open-ended, with only some core restraints. This project emphasized the importance of adaptable design, and making a program that could grow and change without needing complete restructuring.
